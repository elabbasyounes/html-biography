  les balises html utilise

<!DOCTYPE HTML>: Tous les documents HTML doivent commencer par une déclaration <!DOCTYPE>.
La déclaration n'est pas une balise HTML. Il s'agit d'une "information" pour le navigateur 
sur le type de document auquel il doit s'attendre.

<HTML>: La balise <html> est le conteneur de tous les autres éléments HTML

<title>: cette balise définit le titre du document

<meta>: cette balise définit les métadonnées d'un document HTML.

<link>: cette balise définit la relation entre le document actuel et une ressource externe
 comme une feuille de style

<head>: L'élément <head> est un conteneur de métadonnées (données sur les données) 
et est placé entre la balise <html> et la balise <body>.

<body>: cette balise définit le corps du document.

<div>: cette balise définit une division ou une section dans un document HTML.

<span>: span est un conteneur en ligne utilisé pour baliser une partie d'un texte ou une partie d'un document.

h1/h4: utilisé pour définir les en-têtes HTML.

<img>: utilisé pour inserer une image

<p>: utilisé pour définir une paragraph

<ul>: cette balise définit une liste non ordonnée.

<li>: cette balise définit un élément de liste.

<a>: cette balise définit un lien hypertexte, qui est utilisé pour lier d'une page à une autre.
